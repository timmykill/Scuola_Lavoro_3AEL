# MANZ AG

La MANZ AG è una azienda manufatturiera tedesca che produce Pannelli Fotovoltaici, materiali vetrosi per la creazione di schermi touch screen e macchinari per la produzione delle batterie al litio.
Ha circa 2000 dipendenzi nel mondo e lo stabilimento produttivo in'Italia è locato a Sasso Marconi.
